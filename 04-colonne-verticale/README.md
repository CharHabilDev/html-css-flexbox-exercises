# Exercice 4. Colonne verticale

## Objectif 

Crée une carte contenant :

* un titre
* un paragraphe
* un bouton

Les éléments doivent être empilés verticalement avec un espace entre eux.

## Compétences travaillées
- `display: flex`
- `flex-direction`
- `gap`

## Structure HTML

````html
<div class="card">
    <h1>Titre</h1>
    <p>Paragraphe</p>
    <button>Bouton</button>
</div>
````

## Contraintes

* les éléments doivent être affichés en colonne
* il doit y avoir un espace régulier entre eux
* uiliser Flexbox
