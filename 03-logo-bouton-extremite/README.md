# Exercice 3. Deux blocs aux extrémités

## Objectif

Créer un en-tête contenant un logo aligné à gauche et un bouton aligné à droite.

## Compétences travaillées
- `display: flex`
- `justify-content`
- `align-items`

## Structure HTML

```html
<header>
  <div class="logo">Logo</div>
  <button>Connexion</button>
</header>
```

## Contraintes

* le logo doit être complètement à gauche
* le bouton doit être placé complètement à droite
* les deux éléments doivent être alignés verticalement au centre
* utiliser Flexbox
