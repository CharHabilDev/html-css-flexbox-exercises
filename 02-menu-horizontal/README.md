# Exercice 2. Menu horizontal

## Objectif
Créer une barre de navigation avec 4 liens alignés sur une seule ligne.

## Compétences travaillées
- `display: flex`
- `justify-content`
- `align-items`
- `gap`

## Structure HTML
```html
<nav>
  <a href="#">Accueil</a>
  <a href="#">Services</a>
  <a href="#">Blog</a>
  <a href="#">Contact</a>
</nav>
```

## Contraintes

* les liens doivent être affichés sur une seule ligne
* il doit y avoir un espace régulier entre eux
* les liens doivent être centrés verticalement dans la barre de navigation
* utiliser Flexbox
