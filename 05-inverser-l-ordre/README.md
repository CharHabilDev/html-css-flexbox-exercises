# Exercice 5. Inverser l’ordre

## Objectif

Créer plusieurs versions d’un même conteneur pour observer l’effet de `flex-direction` sur l’ordre et la direction des éléments.

## Compétences travaillées
- `display: flex`
- `flex-direction`

## Structure HTML

```html
<div class="container">
  <div>1</div>
  <div>2</div>
  <div>3</div>
</div>
```

## Contraintes

Créer 4 versions :

1. ordre horizontal normal
2. ordre horizontal inversé
3. ordre vertical normal
4. ordre vertical inversé

Utiliser Flexbox pour chaque version.
