# Exercice 1. Centrer un bloc

## Objectif 

Crée une page avec une `div` centrée exactement au milieu de l'écran.

## Compétences travaillées
- `display: flex`
- `justify-content`
- `align-items`
- gestion de la hauteur de l'écran

## Structure HTML :
```html
<body>
  <div class="box">Bloc</div>
</body>
```

## Contraintes

* le `body` prend toute la hauteur de l’écran
* la boîte mesure `200px x 100px`
* le centrage doit être horizontal et vertical
* utiliser Flexbox
